Architecture

Role-Based Access Control

ARCx implements OpenZeppelin's AccessControl for secure permission management:

  • ADMIN_ROLE - Contract administration and bridge configuration
  • MINTER_ROLE - Token distribution during genesis phase
  • PAUSER_ROLE - Emergency pause/unpause controls

Constitutional Guarantees

  • Supply Cap: Immutable maximum supply, no inflation mechanism
  • Minting: One-time finalization, irreversible after completion
  • Migration: Burn-to-migrate mechanism via constitutional bridge
  • Transparency: All allocation schedules publicly verifiable
contract ARCxToken is ERC20, ERC20Capped, AccessControl, Pausable {
    bytes32 public constant ADMIN_ROLE = keccak256("ADMIN_ROLE");
    bytes32 public constant MINTER_ROLE = keccak256("MINTER_ROLE");
    bytes32 public constant PAUSER_ROLE = keccak256("PAUSER_ROLE");
    
    constructor() ERC20("ARCx", "ARCx") ERC20Capped(100_000_000 * 10**18) {
        _grantRole(DEFAULT_ADMIN_ROLE, msg.sender);
        _grantRole(ADMIN_ROLE, msg.sender);
        _grantRole(MINTER_ROLE, msg.sender);
        _grantRole(PAUSER_ROLE, msg.sender);
    }
}

Migration Protocol

ARCx to FUEL Bridge

Constitutional bridge protocol enables transparent migration from ARCx to FUEL tokens when the ARC network launches. The bridge implements a one-way burn mechanism with cryptographic proof and auditable conversion.

Migration Process

  • Bridge Activation: Community governance vote to activate migration
  • Burn Mechanism: ARCx tokens permanently burned during conversion
  • FUEL Allocation: Equivalent FUEL tokens allocated on ARC network
  • Cryptographic Proof: Merkle proof system for transparent verification
  • Audit Trail: Complete on-chain record of all migrations
function migrateTo_FUEL(uint256 amount, bytes32[] calldata merkleProof) external {
    require(migrationActive, "Migration not yet activated");
    require(amount > 0, "Amount must be greater than zero");
    
    // Burn ARCx tokens
    _burn(msg.sender, amount);
    
    // Record migration with cryptographic proof
    emit TokenMigration(msg.sender, amount, block.timestamp, merkleProof);
}
